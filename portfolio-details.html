CROB Website

  • CROB.SHOP is a creative and Subtle Design website for a Posters & Design Company
  • It is Designed to attract Customers to its simple and clear thought of design, It features Basic functionality for display of products and other features
  • It is styled with Tailwindcss that enhances the visual effects and design of the website

CROB Backend

  • The Crob Website backend is developed to handle the orders placed and retrive the required information and also handle some form API's

  • The order details are displayed on the web interface.

  • It handles the status of all orders placed by a user and update it to when requested from the frontend as per the status of thier order in database Dynamically. for this we use mongodb queries and js on frontend side
  • The TECH used are Python, Flask, MongoDB.

SuperMarket TurtleBot

  • This TurtleBot was designed with its open source Bot resources.
  • It is designed to navigate through the layout of a supermarket and carry out simple tasks like placing of items on rack.
  • It has incorporated Lidar Technology for navigation and Obstacle avoidance
  • It uses different sensors like ultrasonic sensor, Acceleration Sensor, IR sensor to place the objects properly on rack as programmed.
  • It has ESP32 to allow control and management of the device and data.
